🚲 Bicycles Store – SQL Project

📝 Project Overview

This SQL-based project models a bike store’s database to manage key business components such as customers, staff, stores, brands, categories, and inventory. The goal is to execute queries to support operations like sales tracking, product analysis, and staff performance.

🗂 Entity-Relationship Diagram

🛠 Technologies & Skills Applied

  • Database schema creation (DDL)
  • Data retrieval with advanced SQL queries (DQL)
  • Joins, aggregations, filtering, ordering, grouping
  • Data modeling and normalization
